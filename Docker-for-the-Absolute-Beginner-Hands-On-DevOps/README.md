# Docker for the Absolute Beginner - Hands On - DevOps

## Course Overview
Hands-on Docker training for beginners covering containerization fundamentals and DevOps practices.

## Course Structure

### 01-Introduction
- What is Docker?
- Why use containers?
- Docker installation
- Docker architecture

### 02-Docker-Commands
- Basic Docker commands
- Container lifecycle management
- Docker CLI essentials

### 03-Docker-Run
- Running containers
- Port mapping
- Environment variables
- Volume mounting

### 04-Docker-Images
- Understanding Docker images
- Creating Dockerfiles
- Building custom images
- Image layers and caching

### 05-Docker-Compose
- Multi-container applications
- docker-compose.yml syntax
- Service orchestration
- Networking between containers

### 06-Docker-Registry
- Docker Hub
- Private registries
- Pushing and pulling images
- Image tagging strategies

### 07-Docker-Engine-Storage-Networking
- Docker engine architecture
- Storage drivers
- Volume management
- Network types and drivers

### 08-Container-Orchestration
- Introduction to orchestration
- Docker Swarm basics
- Kubernetes overview
- Scaling containers

### 09-Tasks
- Hands-on projects
- Real-world scenarios
- Practice exercises
- Challenge labs
