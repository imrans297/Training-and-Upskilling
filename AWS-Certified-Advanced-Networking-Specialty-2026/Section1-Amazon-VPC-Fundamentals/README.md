# Section 1: Amazon VPC Fundamentals

## Topics Covered

- VPC Overview and Architecture
- CIDR Blocks and IP Addressing
- Subnets (Public and Private)
- Route Tables
- Internet Gateway (IGW)
- NAT Gateway and NAT Instance
- Security Groups
- Network ACLs (NACLs)
- VPC Peering

## Learning Objectives

- Understand VPC architecture and components
- Design and implement VPC with proper CIDR planning
- Configure routing for public and private subnets
- Implement security using Security Groups and NACLs
- Set up VPC peering connections
