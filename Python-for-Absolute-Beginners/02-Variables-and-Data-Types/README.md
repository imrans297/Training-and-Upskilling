# 02. Variables and Data Types

## Variables
Variables store data values.

```python
name = "Imran"
age = 25
height = 5.9
is_student = True
```

## Data Types

### Numbers
```python
# Integer
count = 10
year = 2024

# Float
price = 99.99
temperature = 36.6

# Operations
sum = 10 + 5
difference = 10 - 5
product = 10 * 5
quotient = 10 / 5
power = 2 ** 3  # 8
```

### Strings
```python
name = "Python"
message = 'Hello World'
multiline = """This is
a multiline
string"""

# String operations
full_name = "Imran" + " " + "Shaikh"
repeated = "Ha" * 3  # "HaHaHa"
length = len("Python")  # 6
```

### Booleans
```python
is_active = True
is_deleted = False
```

## Type Conversion
```python
# String to int
age = int("25")

# Int to string
age_str = str(25)

# String to float
price = float("99.99")
```
