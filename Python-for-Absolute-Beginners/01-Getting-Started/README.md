# 01. Getting Started with Python

## What is Python?
Python is a high-level, interpreted programming language known for simplicity and readability.

## Installation

### Linux (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install python3 python3-pip
python3 --version
```

### Verify Installation
```bash
python3 --version
pip3 --version
```

## Your First Program

Create `hello.py`:
```python
print("Hello, World!")
```

Run it:
```bash
python3 hello.py
```

## Python Interactive Shell
```bash
python3
>>> print("Hello")
>>> 2 + 2
>>> exit()
```
