# 10. Projects

Build real-world applications to practice your skills.

## Project 1: CLI Calculator
Build a command-line calculator with basic operations.

**Features:**
- Add, subtract, multiply, divide
- Handle errors (division by zero)
- Keep history of calculations

## Project 2: To-Do List Application
Create a task management system.

**Features:**
- Add tasks
- Mark tasks as complete
- Delete tasks
- Save to file
- Load from file

## Project 3: File Organizer
Organize files in a directory by extension.

**Features:**
- Scan directory
- Group files by type (.pdf, .jpg, .txt, etc.)
- Move files to organized folders
- Generate report

## Project 4: AWS EC2 Instance Manager
Automate AWS EC2 operations using boto3.

**Features:**
- List all EC2 instances
- Start/stop instances
- Get instance details
- Filter by tags

## Project 5: Log File Analyzer
Parse and analyze log files.

**Features:**
- Read log files
- Count error/warning/info messages
- Find most common errors
- Generate summary report

## Getting Started
Each project has its own directory with:
- README.md (requirements and instructions)
- starter code
- solution (hidden until you try)
