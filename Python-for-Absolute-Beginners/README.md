# Python for Absolute Beginners

Complete Python course from zero to building real projects.

## Course Structure

### 01. Getting Started
- Python installation and setup
- IDE/Editor configuration (VS Code)
- First Python program
- Python interactive shell

### 02. Variables and Data Types
- Variables and naming conventions
- Numbers (int, float)
- Strings and string operations
- Booleans
- Type conversion

### 03. Control Flow
- if/elif/else statements
- Comparison and logical operators
- for loops
- while loops
- break and continue

### 04. Functions
- Defining functions
- Parameters and arguments
- Return values
- Scope
- Lambda functions

### 05. Data Structures
- Lists
- Tuples
- Dictionaries
- Sets
- List comprehensions

### 06. File Handling
- Reading files
- Writing files
- Working with CSV
- Working with JSON

### 07. Error Handling
- try/except blocks
- Common exceptions
- Custom exceptions
- finally clause

### 08. Modules and Packages
- Importing modules
- Standard library overview
- Creating modules
- pip and package management

### 09. OOP Basics
- Classes and objects
- Attributes and methods
- Inheritance
- Encapsulation

### 10. Projects
- CLI calculator
- To-do list application
- File organizer
- Web scraper basics
- AWS automation script

## Learning Path

1. Complete modules 01-09 in order
2. Practice exercises in each module
3. Build projects in module 10
4. Apply Python to AWS automation

## Prerequisites
- Basic computer skills
- No programming experience required

## Time Estimate
- 4-6 weeks (1-2 hours daily)
