# Quick Start Guide

## Start Learning Python Today!

### Step 1: Verify Python Installation
```bash
python3 --version
```

If not installed:
```bash
sudo apt update
sudo apt install python3 python3-pip
```

### Step 2: Run Your First Program
```bash
cd Python-for-Absolute-Beginners/01-Getting-Started
python3 hello.py
```

### Step 3: Follow the Learning Path
1. Read `README.md` in each module
2. Study the examples
3. Complete `exercises.py`
4. Move to next module

### Step 4: Practice Daily
- Spend 1-2 hours daily
- Type code yourself
- Experiment and break things
- Learn from errors
