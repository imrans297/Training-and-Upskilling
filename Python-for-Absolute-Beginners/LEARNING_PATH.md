# Python Learning Path

## Week 1: Foundations
- **Day 1-2:** Module 01 - Getting Started
  - Install Python
  - Run first program
  - Explore interactive shell

- **Day 3-4:** Module 02 - Variables and Data Types
  - Practice with different data types
  - Complete all exercises
  - Build simple calculator

- **Day 5-7:** Module 03 - Control Flow
  - Master if/else statements
  - Practice loops
  - Complete exercises

## Week 2: Functions and Data Structures
- **Day 1-3:** Module 04 - Functions
  - Write reusable functions
  - Understand scope
  - Practice with exercises

- **Day 4-7:** Module 05 - Data Structures
  - Master lists and dictionaries
  - Learn list comprehensions
  - Complete exercises

## Week 3: File Handling and Error Management
- **Day 1-3:** Module 06 - File Handling
  - Read/write files
  - Work with CSV and JSON
  - Build file processor

- **Day 4-7:** Module 07 - Error Handling
  - Handle exceptions properly
  - Write robust code
  - Complete exercises

## Week 4: Advanced Topics
- **Day 1-3:** Module 08 - Modules and Packages
  - Use standard library
  - Install packages with pip
  - Create your own modules

- **Day 4-7:** Module 09 - OOP Basics
  - Understand classes and objects
  - Practice inheritance
  - Complete exercises

## Week 5-6: Projects
- **Week 5:** Projects 1-3
  - CLI Calculator
  - To-Do List App
  - File Organizer

- **Week 6:** Projects 4-5
  - AWS EC2 Manager
  - Log File Analyzer

## Daily Practice Tips
1. Code for at least 1 hour daily
2. Type code yourself (don't copy-paste)
3. Experiment and break things
4. Read error messages carefully
5. Use Python documentation
