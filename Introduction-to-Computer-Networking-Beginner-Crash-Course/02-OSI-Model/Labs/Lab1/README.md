# Lab 1: OSI Layer Identification

## Objective
Identify which OSI layer each protocol and device operates at.

## Tasks

### Task 1: Protocol Classification
Assign each protocol to its OSI layer:

**Protocols:**
1. HTTP
2. TCP
3. IP
4. Ethernet
5. DNS
6. ICMP
7. ARP
8. FTP
9. UDP
10. SSL/TLS

### Task 2: Device Layer Mapping
Identify the OSI layer for each device:

1. Hub
2. Switch
3. Router
4. Firewall (can be multiple layers)
5. Load Balancer

### Task 3: Data Unit Identification
Match the data unit with its layer:

**Data Units:** Bits, Frame, Packet, Segment, Data

**Layers:** Physical, Data Link, Network, Transport, Application

### Task 4: Real-World Scenario
You're browsing https://www.example.com

List what happens at each OSI layer from Application (7) down to Physical (1).

## Expected Outcomes
- Identify protocols at each layer
- Understand data encapsulation
- Trace network communication through layers
