# Introduction to Computer Networking - Beginner Crash Course

## Course Overview
Master computer networking fundamentals from scratch. Learn protocols, IP addressing, routing, security, and troubleshooting through hands-on labs.

## Course Structure

### 01. Networking Fundamentals
- What is a network?
- Network types (LAN, WAN, MAN)
- Network topologies
- Network devices (Hub, Switch, Router)

### 02. OSI Model
- 7 layers of OSI model
- Layer functions and protocols
- Data encapsulation
- Layer interactions

### 03. TCP/IP Protocol Suite
- TCP vs UDP
- Three-way handshake
- Port numbers
- Common protocols (HTTP, HTTPS, FTP, SSH)

### 04. IP Addressing & Subnetting
- IPv4 addressing
- Public vs Private IPs
- Subnetting basics
- CIDR notation
- IPv6 introduction

### 05. DNS & DHCP
- DNS resolution process
- DNS record types
- DHCP operation
- DNS troubleshooting

### 06. Routing & Switching
- Routing fundamentals
- Static vs Dynamic routing
- Switching concepts
- VLANs basics

### 07. Network Security Basics
- Firewalls
- Security groups
- VPN basics
- Common attacks and prevention

### 08. Troubleshooting Tools
- ping, traceroute, nslookup
- netstat, ss, ip commands
- Wireshark basics
- Common issues and solutions
