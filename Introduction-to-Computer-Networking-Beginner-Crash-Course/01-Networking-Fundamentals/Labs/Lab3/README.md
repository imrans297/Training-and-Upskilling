# Lab 3: Network Types (LAN, WAN, MAN)

## Objective
Differentiate between LAN, WAN, and MAN networks.

## Tasks

### Task 1: Network Classification
Classify each scenario as LAN, WAN, or MAN:

1. Office building with 5 floors, all computers connected
2. Company with offices in New York, London, and Tokyo
3. University campus with multiple buildings
4. City-wide public WiFi network
5. Home network with WiFi
6. Bank's ATM network across a country

### Task 2: Network Planning
You're an IT consultant. Design networks for:

**Client A:** Local coffee shop
- 1 location
- 10 devices
- Guest WiFi needed

**Client B:** Regional retail chain
- 15 stores across 3 cities
- Centralized inventory system
- Each store has 5-10 devices

**Client C:** City government
- 20 buildings across city
- Shared resources
- High security required

For each, specify:
- Network type(s)
- Key devices needed
- Approximate coverage area

### Task 3: Performance Analysis
Compare LAN vs WAN:
- Typical speed
- Latency
- Cost
- Ownership
- Security considerations

## Expected Outcomes
- Distinguish between network types
- Understand appropriate use cases
- Recognize performance characteristics

**Task 1:**
1. LAN

3. LAN/MAN (depending on size)
4. MAN
